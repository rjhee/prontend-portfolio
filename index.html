#밤새도록 #개발이 재밌는 # 1일1커밋 # 피땀눈물 # 성장하는 개발자

"안녕하세요 신입 개발자입니다"

직장동료에게 엑셀이 재밌다고 했더니 저에게 잘 맞을거 같다며 코딩과 개발자라는 직업을 알려주었습니다.
개발자가 어떤일을 하는지 전혀 몰라서 알아보게 되었고, 작은 프로젝트를 만들어 보게 되었습니다.
배우는 과정은 어려웠지만 문제가 생기고, 생각하고 그것을 논리적으로 해결하는 과정이 저에게 너무 매력적 이었습니다.
본격적으로 배우고자 학원을 다니게 되었고, 프론트엔트 개발자로 성장하기 위해선 더 많은 정보와 지식들이 필요해 유료온라인 강의를 들으며 배웠습니다.

개발이 너무 재밌어서 밤새 프로젝트를 만들기도 하고,
너무 피곤해서 자려고 누웠을땐 아까 해결하지 못한 버그를 어떻게 해결할 수 있을지 자면서 생각하다 꿈에서 까지 코딩한 적도 많았습니다.
버그가 해결이 안되면 해결이 될때 까지 밥먹으면서도, 씻으면 서도, 길을 걸어 가면서도 생각을 멈추지 못하고
방법이 떠오르면 메모장에 적어두고 다음날 아침 일어나자 마자 적용하면서 시간가는 줄 모르게 재밌게 개발 공부를 했었습니다.

즐기는 사람을 이길수 없다는 말 처럼, 개발을 즐기는 사람은 언젠간 좋은 결과물을 만들어 낸다고 생각합니다.

"저는 지금 이것을 할 수 있어요!"

c# 언어로 간단한 숫자야구 콘솔 게임과 디지털도어록, 계산기 등을 만들어 보게 되었습니다.
숫자야구 콘솔 게임을 만들 때, 사용자가 입력한 값이 정답 중에 있는지? 있다면 또 다른 조건도 일치하는지? 입력한 값에 따라서 몇 가지 조건을 충족했는지 사용자에게 알림 창을 띄워야 하는데,
어떤 조건을 먼저 사용해야 되는지, 충족한 조건을 알림 창에 어떻게 띄우는지 등 만드는 과정 중 문제가 생겼을 때 그 문제를 해결하기 위해서 필요한 컴퓨터적인 사고 방식을 배우는 것이 재밌었습니다.

그 후로 HTML과 CSS를 공부하기 위해서 매일 노트 정리하면서 온라인 동영상 강의도 수강하고, 국비지원 학원도 다니게 되었고,
직접 만들어보면서 저에게 가장 어려웠던 HTML 문서를 의미에 맞게 작성하는 연습과 CSS 레이아웃 배치, 반응형 웹 등을 만드는 연습을 하게 되었습니다.
그리고 순수 자바스크립트의 기본 문법과, 웹 APIs를 익히기 위해 이론을 노트 정리하면서 공부하고, 여러 프로젝트를 만들면서 연습했습니다.

또 구현하는 것도 중요하지만 더 성능이 좋은 웹사이트 만들어 보고 싶어서 브라우저, window loading, 이벤트 루프 등을 공부하고, 깃허브에 업로드하면서 관리했습니다.
그리고 프런트엔드 개발자들이 가장 많이 사용하는 리액트를 배우고 블로그 웹앱과 중고마켓 웹앱을 만들어보기도 했습니다.
리액트는 기존에 DOM을 통해서 조작했던 자바 크립트와는 또 다른 방법인 가상 DOM을 사용하는 것과, 변수 대신 state를 통해 데이터를 관리한다는 것, prop를 통해서 데이터를 주고받는 것 등이 너무 생소해 어려웠지만,
이해하기 위해서 다른 사람의 코드를 따라치지 않고, 직접 생각하고 코드를 작성하고 디버깅했고, 메모장 같은 간단한 웹앱을 만들면서 연습하고, 익숙해지기 위해 반복했습니다.
익숙해지니 순수 자바스크립트보다 리액트로 구현하는 것이 유지 보수가 쉽고, 재사용성도 좋고 가상 DOM을 사용하니 빠르고 효과적이게 HTML을 렌더링 하여 사용자들에게 좋은 경험을 주게 되는 것 등등 배우게 되었습니다.

  • 90%

    HTML

    웹표준, 웹 접근성을 고려한 시멘틱 마크업으로 장애인 고령자등도 비장애인과 동등하게 접근하고 이해할 수 있도록 구성할 수 있습니다.

  • 90%

    CSS

    디자이너의 시안 그대로 레이아웃을 배치하고, Media Query를 통해 다양한 디바이스에서도 대응 할 수 있는 반응형으로 웹을 제작할 수 있습니다.

  • 80%

    JavaScript

    바닐라 자바스크립트의 기본문법을 익히고 DOM객체를 제어 할 수 있습니다. 또 브라우저의 원리와 윈도우 로드의 기본을 배워 사용자들에게 빠르고 좋은 경험을 줄 수 있습니다.

  • 80%

    jQuery

    jQuery 문법으로 DOM객체를 제어하여 다양한 애니메이션으로 빠르게 인터렉티브한 웹을 제작할 수 있습니다.

  • 70%

    React

    바닐라자바스크립트로 만든 웹사이트를 유지보수하는데 한계를 느껴 배우게 되었습니다. 하나의 HTML로 직접 DOM객체를 제어하지 않아 사용성이 빠르고, 유지보수가 효율적인 웹을 제작할 수 있습니다.

  • 70%

    Sass/SCSS

    Sass와 SCSS를 사용하여 CSS를 더욱더 빠르고, 가독성이 좋으며 유지보수하기 편하도록 웹을 제작할 수 있습니다.

  • 60%

    Vue.js

    리액트보다 배우기 쉽다는 장점으로 배우게 되었습니다. 리액트와는 어떤 차이가 있는지 현재 배우고 있는 중 입니다.

당근뽑기 게임 (콘솔 게임)

👉🏼 완성작 보러가기
컨셉
벌레를 피해서 당근을 제한시간내에 다 뽑아야 한다. 제한시간안에 당근을 다 뽑으면 다음 레벨로 넘어갈 수 있다. 벌레를 클릭하면 게임이 끝난다.
사용한 기술
기능

- 당근을 제한시간내에 다 뽑으면 레벨업.
